Clinical Data Tools

医生库、药品库、医院库

三类基础库负责回答“谁、在哪、用什么”。它们不单独成产品，而是给搜索、问答、挂号和医院 AI 工作流提供可追溯数据。

Doctor DB医生库

按姓名、医院、科室、疾病方向、论文、任职和医生排名筛选，支持医生主页自动补全和合作主题推荐。

主任医师儿科智慧医院论文 150+
样例查询 doctor?hospital=CUHK-Shenzhen&tag=互联网医疗
进入医生库样例
Drug DB药品库

聚合通用名、商品名、适应症、禁忌、相互作用、医保属性和说明书摘要，用于临床问答前的来源校验。

适应症禁忌相互作用医保
样例查询 drug?keyword=阿莫西林&field=interaction
查看 API 接入方式
Hospital DB医院库

支持按地区、等级、专科能力、挂号渠道和 AI 合作方向检索，默认展示深圳龙岗区医院样例。

龙岗区三甲/三级专科能力可认领
样例查询 hospital?city=深圳&district=龙岗区&service=挂号
