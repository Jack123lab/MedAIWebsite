# Freedom AI Academic Community

此目录是 Freedom AI 的学术社区门户 MVP，位置为：

本目录是 GitHub Pages 发布根目录。

## 页面

- `index.html`：初始登录页
- `home.html`：登录后的站内首页
- `profile.html`：个人页面
- `tutorials.html`：医学 AI 课程
- `community.html`：临床与科研讨论
- `research.html`：成果收录
- `demos.html`：工具示例
- `contribute.html`：投稿与收录

## 静态资源

- `assets/css/style.css`：统一学术风样式
- `assets/js/main.js`：列表筛选交互
- `assets/images/academic-network.svg`：首页视觉资产

## GitHub Pages 部署建议

站点文件已经位于仓库根目录，可直接作为 GitHub Pages 静态站点发布。

## 内容维护建议

后续教程、Demo、成果和 Prompt 模板可以逐步放入 `content/` 下，以 Markdown 形式沉淀，再由页面链接到对应内容。
